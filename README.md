# Rob

Rob is an implementation of Eli Bendersky's Bob Scheme in Ruby. It's not useful,
but it's fun :)

## Installation

Add this line to your application's Gemfile:

    gem 'rob'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rob

## Usage

Don't use it.
